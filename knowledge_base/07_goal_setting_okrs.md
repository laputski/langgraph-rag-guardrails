# Goal Setting and OKRs

Acme Corp uses the **OKR (Objectives and Key Results)** framework to align individual, team, and company goals.

## OKR Cycle

OKRs are set quarterly:
- **Q1 OKRs**: Set by January 15th
- **Q2 OKRs**: Set by April 15th
- **Q3 OKRs**: Set by July 15th
- **Q4 OKRs**: Set by October 15th

Each employee should have **3–5 Objectives** per quarter, with **2–4 Key Results** per Objective.

## OKR Structure

**Objective**: A qualitative, inspiring goal that defines where you want to go.
**Key Result**: A measurable, time-bound outcome that proves the Objective was achieved.

Example:
- **Objective**: Improve customer satisfaction for the Support team
  - KR1: Increase CSAT score from 72% to 85% by end of quarter
  - KR2: Reduce average response time from 4 hours to 2 hours
  - KR3: Complete 3 process improvement initiatives identified in Q3 retro

## Scoring

Key Results are scored on a 0.0–1.0 scale at the end of each quarter:
- 0.7–1.0 = Green (achieved or exceeded)
- 0.4–0.69 = Yellow (partial progress)
- 0.0–0.39 = Red (insufficient progress)

**A 0.7 score is considered success.** OKRs should be ambitious — scoring 1.0 consistently suggests goals are too easy.

## Individual vs. Company OKRs

- Company OKRs are set by the Leadership Team and published on the intranet by the 10th of the first month of each quarter.
- Team OKRs are set by department heads in alignment with Company OKRs.
- Individual OKRs should be set in conversation with your manager and ladder up to at least one Team OKR.

## OKR Tracking

Use Workday Goals module to log and update your OKRs. Weekly progress updates are encouraged. OKRs are reviewed during the mid-year check-in and annual performance review.
